# Boosts marketing assets

Drop the real Jordan Mills example assets here. The /boosts page and the
resume modal reference these exact paths:

- jordan-mills-visual-boost.png  (Visual Boost infographic)
- jordan-mills-short-boost.mp3   (Short Boost Audio)
- jordan-mills-podcast-boost.mp3 (Podcast Style Boost, still pending)
- jordan-mills-resume.jpg        (ATS Resume, shown in the modal)

Audio must be a plain progressive file (MP3, or a non-fragmented M4A with a
faststart moov). A DASH / fragmented-MP4 export will not play in a native
<audio> element, so re-encode those to MP3 before adding them here.

Until a file is present, the page shows an on-brand placeholder.
